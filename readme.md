#Project for MIPT

Frontend uses node.js (folder: ./hookies)

Backend uses django and simplejwt for authorisation (folder: ./myback)

-------------------------------------------------------------------------------

## For users

To run project, start two consoles:

1) cd ./myback;

python manage.py runserver 8000

2) cd hookies;

npm run start
